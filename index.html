05 — En producción

Proyectos

01 · TFG DAM · Feb–Mar 2026
Desplegado · Azure App Service

7 Wonders Architects

.NET MAUI · ASP.NET Core · SignalR · Azure

Clon multijugador del juego de mesa para duelos online en tiempo real. Lo más interesante del proyecto no es que funcione, sino cómo está construido: cliente y servidor completamente desacoplados, lógica de negocio aislada en un GameManager sin dependencias de UI ni de red, y comunicación bidireccional de baja latencia.

  • Dominio puro en POCOs: GameManager con toda la lógica sin dependencias externas
  • Hub SignalR distribuyendo el estado de partida en JSON entre jugadores
  • ConcurrentDictionary para múltiples salas simultáneas sin bloqueos
  • UI asíncrona con CommunityToolkit.Maui, sincronización segura al hilo principal

Java 17 · JavaFX · MVC

Regicide

Java 17 · JavaFX 20 · MVC · JSON · CSV

Implementación completa del juego de cartas con reskin visual al universo Pokémon. El resultado más interesante es que cambiar la temática visual al completo no requirió modificar ni una línea del modelo de dominio — separación real entre lógica y presentación, no solo en el papel.
